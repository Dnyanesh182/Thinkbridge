Welcome to Web Development

This is Session 1: Introduction. Here you will learn about the fundamentals of HTML, CSS, and JavaScript, and how to start creating your first web pages.

Use semantic HTML to structure your content and prepare for interactive sessions.

HTML Tags Overview

<div>

Defines a section or division in HTML.

<p>

Defines a paragraph of text.

<a>

Defines a hyperlink to another page or section.

<ul>

Defines an unordered list.

<li>

Defines a list item inside a list.

<color>

Sets the color of text.

<background-color>

Sets the background color of an element.

<margin>

Sets the space outside an element.

<padding>

Sets the space inside an element.

<border>

Defines the border of an element.

<let>

Declares a block-scoped variable.

<const>

Declares a block-scoped constant variable.

<function>

Defines a reusable function.

<if>

Executes code based on a condition.

<for>

Loops through code a specific number of times.
